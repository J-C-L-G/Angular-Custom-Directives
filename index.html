Custom Directives

Custom and Reusable Directives for AngularJS

Download this project as a .zip file Download this project as a tar.gz file

Description

Proof of Concept - AngularJS custom directives for multiple purposes such as:
(form validation, text censoring, spread sheets, graphs, infinite image scroll, etc).

This demo application was created using:


Charts

Custom Directive to display data in charts:

  • < graph-directive type='line' >
  • < graph-directive type='column' >
  • < pie-graph-directive >

Form Validation

Custom Directive to perform client side validation:

  • < custom-error-message-directive >
  • < value-match-directive >
  • < ensure-type-directive >

Spreadsheet

Custom Directive to display tabular data and broadcast data changes:

When the data changes in the spreadsheet, the directive emits an event allowing other directives subscribed to this event update its data accordingly.

  • < spread-sheet-directive source='dataInScope' >







Image Scroll

Custom Directive to display images from a Flickr Feed:

  • < infinite-scroll-directive >
